Panhelios SDK

About the SDK

The Panhelios Software Development Kit enables small teams to produce modern browser games. We used the SDK in the production of all our games.
  • Model-driven development to reduce production overhead.
  • Object-relational mapping for easy access of game data.
  • HelioScript translates to PHP and JavaScript code, so all code can run on the server and the client side in sync.
  • Professional IDE: Eclipse
  • Supports WebSocket and AJAX
  • Modern HTML 5 graphics engine
